# Install Vim/GVim from Source

## 1. Vim Source Code
Get [vim source code](https://github.com/sagedemage/vim-config)

## 2. Dependencies
```
sudo apt install make
sudo apt install clang
```

## 3. Build
```
sudo apt install libxt-dev
sudo apt install libgtk-3-dev
make reconfig
```

## 4. Install
```
sudo make install
```
