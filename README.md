# gvim-config
My configuration of vim using the vim plugin manager.

## Setup instructions

Install [vim-plug](https://github.com/junegunn/vim-plug)

Go to the gvim-config folder
```
cd gvim-config
```

Copy this config to the vim configuration file
```
cp gvimrc ~/.gvimrc
```

Install plugins
```
:PluginInstall
```
